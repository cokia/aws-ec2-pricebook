AWS PriceBook
====
Simple Aws Pricebook

Get data from

- [http://a0.awsstatic.com/pricing/1/ec2/linux-od.min.js](http://a0.awsstatic.com/pricing/1/ec2/linux-od.min.js)

Install
---

    npm install

Run
---

    node server.js

Then just point your html5 compliant web browser to `http://127.0.0.1:8080`
